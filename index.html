debswarm apt repository

The official signed APT repository for debswarm — peer-to-peer package distribution for Debian and Ubuntu.

This is an APT repository, not a website: it is meant to be added to sources.list, not browsed. It always serves the current release, for amd64, arm64, and armhf, and is republished automatically on every stable release.

Install

Add the signing key and the repository:

curl -fsSL https://clintcan.github.io/debswarm/debswarm-archive-keyring.gpg \
  | sudo tee /usr/share/keyrings/debswarm-archive-keyring.gpg > /dev/null

sudo tee /etc/apt/sources.list.d/debswarm.sources > /dev/null <<'EOF'
Types: deb
URIs: https://clintcan.github.io/debswarm
Suites: stable
Components: main
Signed-By: /usr/share/keyrings/debswarm-archive-keyring.gpg
EOF

Then install. The package creates the debswarm system user, enables and starts the systemd service, and drops in the APT proxy config — so APT starts going through debswarm immediately:

sudo apt-get update
sudo apt-get install debswarm

Verify the signing key

Before trusting any third-party repository, check the key you just installed matches the fingerprint below:

gpg --show-keys --with-fingerprint /usr/share/keyrings/debswarm-archive-keyring.gpg
15EF 0050 5F36 C2DF C844  3CF9 3B75 F449 CA15 E893

RSA-4096, sign-only, used exclusively to sign this repository. Every package is GPG-signed, and apt verifies that signature itself on every apt-get update.

Automatic upgrades

To let unattended-upgrades pick up new debswarm versions:

sudo tee /etc/apt/apt.conf.d/52debswarm-unattended > /dev/null <<'EOF'
Unattended-Upgrade::Origins-Pattern {
  "origin=debswarm,codename=stable";
};
EOF

Other ways to install

  • Container image: docker pull ghcr.io/clintcan/debswarm:latest (multi-arch, distroless)
  • Standalone .debs are attached to each GitHub release
  • From source: see the README
